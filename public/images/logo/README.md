# Логотипы приложения BAR BOSS ONLINE

Эта папка содержит файлы логотипов для разных контекстов использования.

## Структура логотипа:

Логотип состоит из:
- **Иконка**: Клипборд с бар-чартом и стрелкой роста (золотисто-желтый цвет)
- **Текст**: "BAR BOSS" (крупный) и "ONLINE" (меньшего размера)

## Требуемые файлы:

1. **logo-light.png** - Логотип для черного фона (сайдбар/меню)
   - Используется в боковом меню с черным фоном
   - **Цвета**: 
     - Контур клипборда: белый
     - Бар-чарт и стрелка: золотисто-желтый
     - Текст "BAR BOSS" и "ONLINE": золотисто-желтый
   - Фон: черный
   - Размещается автоматически в компоненте AppLogo при использовании в сайдбаре

2. **logo-dark.png** - Логотип для серого фона (главная страница)
   - Используется на главной странице с серым фоном
   - **Цвета**:
     - Контур клипборда: темно-серый/черный
     - Бар-чарт и стрелка: золотисто-желтый
     - Текст "BAR BOSS" и "ONLINE": темно-серый/черный
   - Фон: светло-серый
   - Размещается автоматически на страницах логина, регистрации и главной странице

## Рекомендуемые размеры:

- **Соотношение сторон**: примерно 3:1 (ширина к высоте)
- **Стандартный размер**: 
  - Ширина: ~150px (высота: ~50px)
  - Для Retina дисплеев: ~300px ширины (высота: ~100px)
- **Формат**: PNG
- **Фон**: 
  - logo-light.png - черный фон
  - logo-dark.png - светло-серый фон

## Размещение файлов:

Поместите ваши файлы логотипов в эту папку:
- `public/images/logo/logo-light.png` - для черного фона (меню/сайдбар)
- `public/images/logo/logo-dark.png` - для серого фона (главная страница)

## Автоматический выбор:

Компонент `AppLogo` автоматически выберет правильный вариант в зависимости от контекста:
- Если используется в сайдбаре (className содержит 'text-sidebar') → `logo-light.png`
- Во всех остальных случаях → `logo-dark.png`

## Использование компонента:

```tsx
// Стандартное использование (автоматический выбор)
<AppLogo />

// Явное указание варианта
<AppLogo variant="light" />  // Принудительно светлый вариант (для черного фона)
<AppLogo variant="dark" />   // Принудительно темный вариант (для серого фона)

// Размеры
<AppLogo size="small" />   // Компактный размер (h-6)
<AppLogo size="default" /> // Стандартный размер (h-8) - по умолчанию
<AppLogo size="large" />  // Крупный размер (h-12)
```

## Примечания:

- Компонент автоматически подстраивает размеры, сохраняя пропорции
- Логотип содержит текст "BAR BOSS ONLINE", поэтому требуется достаточная ширина для читаемости
- Рекомендуется использовать изображения высокого разрешения для четкости на всех устройствах
